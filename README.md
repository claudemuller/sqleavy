# SQLeavy

A toy implementation of SQLite in C.

# Requirements

- [gcc](https://gcc.gnu.org/)
- or [clang](https://clang.llvm.org/)
- [make](https://www.gnu.org/software/make/)

# Build

```bash
make build
```

# Run

```bash
make run
```

# Run Tests

```bash
make test
```

# Build a Debug Binary

```bash
make debug-build
```

# Start `lldb` or `gdb` With Debug Binary

```bash
make debug
```
